Docker-Redis

发表于 2017-11-25 | 分类于 docker ， DevOps

下载镜像

docker pull redis

启动命令

docker run -d -p 6379:6379 -v /VOLUME:/data –name redis {imageID} redis-server –requirepass {password}

修改相应的imageID 和 password 即可启动一个redis实例
